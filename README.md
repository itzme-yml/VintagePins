# 📌 VintagePins

Um mural digital retrô para salvar e organizar suas inspirações visuais, com direito a um player de música integrado do YouTube!

![VintagePins Preview](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen)

---

## 🎨 Sobre o Projeto

O **VintagePins** é um projeto web inspirado na estética vintage/mural de fotos, onde o usuário pode adicionar novos cards (pins) com imagem e título, pesquisar pelos pins salvos e até escutar músicas pelo YouTube enquanto navega.

---

## ✨ Funcionalidades

* 🖼️ **Adicionar Pins:** Faça upload de imagens locais e adicione um título customizado.
* 💾 **Persistência de Dados:** Os pins ficam salvos no navegador usando `localStorage` (com compressão automática de imagem).
* 🔍 **Busca em Tempo Real:** Filtre seus pins salvos pelo título instantaneamente.
* 🗑️ **Exclusão:** Remova facilmente os pins que não quiser mais.
* 🎵 **Player de Música Integrado:** Cole um link do YouTube para ouvir suas músicas favoritas enquanto organiza o mural.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** (Estruturação semântica)
* **CSS3** (Estilização customizada e responsiva)
* **JavaScript (ES6+)** (Manipulação de DOM, Canvas para compressão e integração com YouTube API)

---

## 🚀 Como Executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Aproveite para criar seus pins!
